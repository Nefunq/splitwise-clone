**Inspired by the original Splitwise(https://www.splitwise.com/) app.**

This is a simple web app where you can split expenses with friends.
You can create groups, add bills, split them equally, see who owes whom, and settle up, all without any hassle. It’s built with PHP and MySQL on the back end, and plain HTML, CSS, and JavaScript on the front end.

**Setup Instructions**

Install XAMPP/WAMP/MAMP with PHP 7.4+ and MySQL (Windows is recommended).
Clone the repository.
Update database credentials in "config.php" if needed.
Move repo to folder: C:\xampp\htdocs\
If you are using Linux 
    
**Import database:**

Start Apache and MySQL.
Open phpMyAdmin.
Create new database "splitwise_clone".
Run the "database.sql" script.
Access the application: http://localhost/splitwise-clone/
